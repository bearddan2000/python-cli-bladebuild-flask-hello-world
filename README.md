# python-cli-bladebuild-flask-hello-world

## Description
A POC for blade build system with dependencies.

## Tech stack
- python
    - flask
- g++
- ninja
- blade

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
